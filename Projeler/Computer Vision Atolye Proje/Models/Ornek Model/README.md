## Proje Bitirme Çalışması
Açıklamada Tüm adımların veri seti veri ön işleme model eğitim test değerlendirmelerin içeriği olmalı,
değerlendirmeler test verisinde Accuracy, precision, recall ve F1-Makro skordan olmalı. Model eğitimden sonra
ağırlık dosyasını indirmelisiniz ve başka kullanıcıların test etmesi için sizin modelinizi nasıl kurup ağırlık dosyasınızı nasıl çekip test edebilir
bunuda sistematik şekilde açıklamalısınız.
